# GUIA RÁPIDO - Repertório da Igreja

## ⚡ Instalação Rápida (5 minutos)

### Passo 1: Preparar
```bash
# Abra o terminal/CMD na pasta do projeto
cd repertorio-igreja
```

### Passo 2: Instalar dependências
```bash
npm install
```
(Espere completar, vai baixar ~500MB)

### Passo 3: Rodar
```bash
npm run dev
```

O navegador abrirá automaticamente em `http://localhost:5173`

## ✅ Primeiro uso

1. **Configurações** → Defina o nome da sua igreja
2. **Cadastrar Hino** → Adicione alguns hinos
3. **Harpa** → Cadastre hinos da Harpa Cristã
4. **Montar Repertório** → Crie seu primeiro repertório
5. **Gerar PDF** → Baixe e veja o resultado

## 🎯 Funções Principais

| O que fazer | Onde ir |
|---|---|
| Adicionar hino | Cadastrar Hino |
| Adicionar Harpa | Hinos da Harpa |
| Criar culto | Montar Repertório |
| Ver/editar cultos | Repertórios Salvos |
| Configurar | Configurações |

## 🔒 Dados Salvos Automaticamente

✅ Tudo é salvo no navegador (sem internet necessária)
✅ Feche e abra - os dados continuam lá
✅ Troque de computador? Exporte em Configurações

## 📱 Em Qualquer Dispositivo

- Computador: http://localhost:5173
- Celular na mesma rede: http://[seu-ip]:5173
- Ou publique na web com `npm run build`

## 🆘 Problemas?

**npm: comando não encontrado**
→ Instale Node.js em https://nodejs.org

**Porta 5173 já está em uso**
→ Feche outro Vite, ou use: `npm run dev -- --port 3000`

**PDFs não funcionam**
→ Verifique se pop-ups estão bloqueados no navegador
